前言

起因

由于某些原因，国内不能访问例如Google、Facebook、Youtube等国外站点。因此我们的学习生活娱乐收到了些许影响。

什么是科学上网

科学上网，通俗的说就是番羽土墙。通过技术手段，能过浏览到被墙住的站点。

我需要科学上网吗？

当你无法忍受百度的竞价排名时，当你忍受不了微博的闲言碎语时，当你无法忍受优酷电视剧插播广告时，说明你确实有必要学科学上网了。
不仅如此你的科研需要Google Scholar查找文献时，当你搜寻领域内的最新资讯时，当你希望了解真实的世界时，这只有科学上网可以帮到你。

科学上网需要很好的技术吗？需要付费吗？

实现科学上网的手段有多种，但是均对计算机技术的要求都很低。复杂的只需要简单的配置即可，简单的甚至只需要打开个软件即可。

技术手段

针对“墙”的不同手段，同样的有不同的“翻”的手段。在此，对一些手段进行简单列举，感兴趣的同学请Google之。不同的技术手段还有对应的工具，下面一次进行介绍。

修改DNS服务器

主要用来反“域名劫持”，比如将你所在运营商提供的DNS服务器服务器改为Google的DNS服务器。此技术存在局限性，简单说，很多站点仍无法进入

修改hosts文件

主要用来反“域名污染”，将hosts文件即可。此技术存在局限性，无法对抗IP黑名单。简单说，有些站点仍无法进入。

加密Web代理

通过代理工具（即一款软件），对网络中传输的数据发送到某个代理服务器，该服务器再发送给你想要访问的网站。

*Lantern（蓝灯）

1下载安装

Lantern是一款开源且跨平台免费的代理工具，其官网为https://getlantern.org/，提供了个平台的下载。

2使用：Lantern

安装之后，你可以通过“开始菜单”或者“桌面上的图标”，来启动Lantern。
然后在任务栏的“托盘区”会出现一个蓝灯的图标；同时，它还会自动弹出系统默认的浏览器，在浏览器上显示它的主界面。这个“主界面”很简单，看一下就明白了。如果你想退出，在任务栏的“托盘区图标”点右键，弹出的“快捷菜单”上会有退出的选项。

3配置参数

主界面的右下角有一个“配置按钮”，点击之后会弹出“Lantern 的配置界面”——这个界面同样很简单，目前只有3个复选框（选项）。其中一个选项是“Proxy all traffic”，俺稍微说一下：如果你勾选了这个选项，那么所有的 HTTP 请求都会走 Lantern 的翻墙通道；反之（没有勾选），那么只有那些“被墙的网站”才会走 Lantern 的线路。Lantern 内置了一个“被屏蔽的网站列表”，以此来判断那些是“被墙的”。

4配置网络软件走 Lantern 的代理

（本章节所说的“网络软件”指的是：浏览器、IM 工具、下载工具……）
Lantern默认会在【本机地址】上开启一个 HTTP 代理的端口，端口号是 8787，如果你的网络软件跟它运行在【同一个操作系统】，那么你只需在网络软件的代理界面上设置 HTTP 代理——地址填写 127.0.0.1 端口号填写 8787（注：“127.0.0.1”表示“本机地址”）
如果你想要【跨操作系统】共享蓝灯的科学上网通道，参见编程随想的一篇博文《多台电脑如何共享番羽土墙通道》。

VPN

VPN是在本地和远程VPN服务器之间建立一个加密通道，由此通道到达目的站点，VPN客户端会在系统中虚拟出一个网卡，之后本机的所有上网应用程序，都通过该虚拟网卡和VPN服务端进行数据传输。
提供VPN服务的是一些VPN服务公司，有些免费，更多的收费。
使用方法,登陆某个服务提供商的官网，下单交费，按照说明使用相应客户端即可。

VPN Gate

由日本的筑波大学开发VPN服务，免费。但对于初次接触的，安装配置略显繁琐。

一些网友共享VPN

如http://free.vpn.wwdhz.com/
一些网友在网站上发布VPN账号，并间隔一段时间更换密码。用户可以根据账号进行配置使用。
VPN账号发布页面

GreenVPN
NydusVPN

等商业VPN，需付费。
其中以Green VPN口碑最佳，笔者使用过安卓版，感觉速度不快，客户端欠美观，Windows客户端不了解。

PAC

PAC其他方式，主要的优点是方便，使用不需要启动依赖组件，不需要对系统进行全局代理，可仅针对特定的站点进行代理。
目前我们所知国内提供PAC收费服务商的主要为熊猫翻滚https://www.pandafan.org/、轻云https://theqingyun.info/等。
熊猫翻滚前一段时间别封，现在改成新域名重新提供服务， 用过一年，速度可以，使用很方便。轻云用过一个月，在笔者的笔记本及CMCC-EDU环境下，体验极差。

Shadowsocks

Shadowsocks方式是一种稳定，使用方便的科学上网方式，越来越多的人开始使用shadowsocks，越来越多的公司提供shadowsocks服务。
推荐一个付费 Shadowsocks 服务提供商 Shadowsock.la。一年一百元左右，非常划算，强烈推荐。
